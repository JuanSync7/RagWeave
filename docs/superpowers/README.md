# docs/superpowers

## Overview

This directory contains implementation plans, design specifications, and skill reference documents produced during development using the Superpowers skill system.

## Subdirectories

| Directory | Purpose |
| --- | --- |
| `plans/` | Implementation plans generated by the `write-implementation` and `writing-plans` skills |
| `specs/` | Design specification documents generated by the `brainstorming` and `write-design` skills |
| `skills/` | Skill reference documents and templates used during development |
