A few common culprits when embeddings look correct but retrieval is off:

  • Chunk size mismatch — query embedding is short but chunks are long paragraphs, skewing cosine similarity.
  • Missing metadata filter — add a source filter to narrow the search space.
  • Top-k too high — pulling 20 chunks adds noise; try top-5 with a reranker.
