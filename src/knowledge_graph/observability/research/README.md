# observability/research

This directory stores the iteration history for the observability coverage research run. It tracks what changed each iteration, the score delta, and the final outcome.

## Contents

| Path | Purpose |
| --- | --- |
| `iterations.tsv` | Tab-separated log of each iteration: commit, hypothesis, score, status, and change summary |
| `changelog.md` | Narrative summary of what improved, what failed, and remaining gaps |
