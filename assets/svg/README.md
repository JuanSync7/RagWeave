# assets/svg/

Reusable scalable SVG components intended for embedding in UI templates or
documentation. Files in this directory carry no fixed width/height and scale
to any container. Each file includes inline usage notes.

## Contents

| Path | Purpose |
| --- | --- |
| `bee.svg` | Reusable bee icon component — origin-centered, scales to any size, includes color palette and usage notes |
