# assets/banners/

Decorative project banner images in SVG format. All banners share a 1280×320 px
viewport and are designed for use in README headers and documentation pages.
Four distinct visual themes are provided, covering both dark and light palettes.

| --- | --- |

| `02-neural-loom.svg` | Light-theme banner — neural loom illustration |
| `03-constellation-pipeline.svg` | Dark-theme banner — constellation/pipeline design |
| `04-woven-hexagon-mesh.svg` | Dark-theme banner — woven hexagon mesh (static) |
| `04-woven-hexagon-mesh-animated.svg` | Dark-theme banner — woven hexagon mesh with CSS animation (used in root README) |
